### Same uuid pv
```bash
vgscan
vgimportclone /dev/xvdb3
vgscan
vgchange -ay  # activates the renamed VG (e.g., ubuntu-vg-clone)
lvscan
# debug
sudo blkid /dev/mapper/ubuntu--vg1-ubuntu--lv
sudo dmesg | tail -n 60
# end
sudo mount -t xfs -o ro,nouuid /dev/mapper/ubuntu--vg1-ubuntu--lv /old_data # mount without uuid
```
